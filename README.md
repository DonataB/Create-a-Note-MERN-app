# Create-a-Note-MERN-app

MERN application to create notes and display them on page.

Link to the newest version: https://create-a-notes.herokuapp.com/


### Technologies

---

This project was executed using the following technologies:

- TypeScript
- React
- Node.js
- Express
- MongoDB
- Styled Components
- GIT
